Meso-Scopic Analysis of Concrete

Establishing meso-scopic model in Abaqus has always been a tough work. This blog would be a brief introduction about my work on an easy way to establish meso-scopic model of concrete.

Several Abaqus-based-Python-combined analysis projects are proposed.

2D Random Particle Model

Intuitive, comparably time-saving, most used in my research and still under developing. This program can generate arbitrary rectangular 2-d random particle section with/without reinforcement in Abaqus. This program is ready for conducting parametric analysis in Abaqus.

3D Random Particle Model

Usually there are two types of partition method, topological partition and geometric partition, currently only topological partition method is provided, each mesh element in the model would be assigned with different material properties to simulate different components of concrete.
